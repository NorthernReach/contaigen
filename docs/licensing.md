
The full license text is in the repository root at `LICENSE`.

## What This Means

GPLv3 allows anyone to use, study, modify, and redistribute Contaigen, including
commercially, as long as they follow the GPLv3 terms.

In practical terms:

- Users may run Contaigen for personal, research, consulting, or commercial work.
- Redistributors must preserve copyright and license notices.
- Modified versions that are distributed must provide corresponding source code.
- Derivative works distributed to others must be licensed under GPLv3-compatible
  terms.
- Proprietary redistribution or closed-source product integration requires
  separate permission from the copyright holder.

## Commercial Licensing

The project can still offer separate commercial licensing in the future for
organizations that want to integrate Contaigen into proprietary products without
GPLv3 redistribution obligations.

Before accepting substantial outside contributions, add contributor terms that
make the licensing model clear. Without contributor terms, future commercial
relicensing can become difficult because contributors retain copyright in their
own contributions.

This document is a project summary, not legal advice. The `LICENSE` file is the
binding license text.
